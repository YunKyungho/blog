JavaScript를 확장한 문법이며 HTML에서 사용한 문법과 흡사한 문법을 사용하여 요소를 생성한다.

```jsx
const Title = <h3 id="title" onMouseEnter={() => console.log("mouse enter")}>Hello</h3>
```

진짜 그냥 html 처럼 만든다. 모든 property들은 원래 html에 적던 대로 넣어주면 된다.

그리고 변수의 첫글자는 꼭 대문자로 해줘야 한다.

이제 생성한 저 Title을 보내기 위해서 함수로 만들어줘야 한다.

```jsx
function Title(){
	return( 
		<h3 id="title" onMouseEnter={() => console.log("mouse enter")}>Hello</h3>
	);
}
// 위 방식 혹은
const Title = () => (
	<h3 id="title" onMouseEnter={() => console.log("mouse enter")}>Hello</h3>
);
// 위 처럼 arrow function으로 함수를 만든다.
```

이런식으로 위에 Title 변수에 담았던 내용을 function Title의 return으로 사용한다.

변수 첫 글자를 대문자로 해야 하는 중요한 이유가 바로

```jsx
const Container = () => (
	<div>
		<Title />
		<Button /> // 버튼은 그냥 있다고 치고
	</div>
);
ReactDOM.render(<Container />, root);
// root는 html에서 불러온 div를 담은 변수
```

이렇게 다른 상위 객체에 담아서 render를 시킬 때 대문자가 아닌 소문자로 <title /> 이런식으로 작성을 하게 될 시 이는 우리가 js에서 만든 요소가 아닌 HTML 요소라고 판단되기 때문이다.
