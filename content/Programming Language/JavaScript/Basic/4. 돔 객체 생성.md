
좀 더 역동적이고 상호작용이 있는 웹사이트를 만들기 위해선 JavaScript에서 돔 객체를 직접 생성해 html로 보내줘야 하는 일이 빈번할 것이다.

그때 쓰이는 것이 createElement와 appendChild이다.

이미지 객체를 만들어 보낸다고 가정하자.

```jsx
const img = document.createElement("img"); //이미지 객체 생성.
img.src = "경로/경로/파일명"; // 이미지 property(경로) 지정.
document.body.appendChild(img); // body로 img 객체를 보냄.
```

그냥 간단한 예시이며 실제로는 어떤 이벤트가 발생했을 때 혹은 랜덤한 img 등을 보여주고 싶을 때 같이 이용할 수 있을 것 같다.

그리고 appendChild 같은 경우 객체가 생성될 상위 객체의 가장 아래에 생성이 되고

prepend를 통하여 객체를 보내줄 수도 있는데 이는 상위 객체의 가장 상단에 객체가 생성된다.
