시계를 만들기 전에 알아둬야 할 2가지 개념이 있다. setInterval과 setTimeout인데 전자에 대해서는 배웠으니 넘어가겠다.

**setTimeout**

---

setTimeout은 setInterval과 사용하는 법은 비슷하다.

```jsx
setInterval(실행할 함수, 반복 주기 지정) //(ms단위)
// 위 처럼 이용했다면
setTimeout(실행할 함수, 얼마나 기다렸다 실행할 것인지 시간 지정) //(ms단위)
```

**new Date()**

---

Js에도 date 객체가 존재하는데 new Date()를 써주면 현재 날짜, 시간을 불러와준다.

```jsx
const date = new Date(); //변수에 date 객체를 담아주고
console.log(`${date.getHours()}:${date.getMinutes()}:${date.getSeconds()}`);
//이렇게 get을 통해 시간 분 초 등을 불러올 수 있다.
```

이런식으로 말이다. (브라우저 콘솔창이라 new Date로 했다.)

그리고 setInterval과 new Date()를 사용해서 반복 주기를 1초로 설정 해주면 1초 마다 시간을 불러올 수 있을 것이다. 그리고 이걸 console.log로 console 창에 찍는 것이 아닌 html 돔 객체의 innerText로 넣어주기만 하면 간단한 시계가 완성된다.

하지만 문제가 있다. 10 미만의 숫자들은 1, 2, 3 이런식으로 한 자리씩 출력이 되며 이건 보기에 썩 좋지 않다. 그래서 사용할 것이 padStart 라는 것이다.

```jsx
const date = new Date();
const hours = String(date.getHours()).padStart(2, "0");
const minutes = String(date.getMinutes()).padStart(2, "0");
const seconds = String(date.getSeconds()).padStart(2, "0");
console.log(`${hours}:${minutes}:${seconds}`);
//padStart는 String에게만 적용이 가능해 형변환을 해줬다.
//padStart(입력한 숫자 만큼의 길이가 되지 않는다면, "이곳에 적은 String을 앞에 채워줘라");
//이게 기본 문법이다.
```

padEnd도 있는데 이건 부족한 자릿수 만큼 뒤에 글자를 추가 해주는 기능이다.
