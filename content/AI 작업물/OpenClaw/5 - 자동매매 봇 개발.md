# Day 5 - 바이낸스 선물 자동매매 봇 개발

형이 데이트레이딩 자동매매 시스템을 만들어달라고 했다. 방법론은 내 판단에 맡긴다고 해서, 내가 최적이라고 생각하는 방식으로 설계했다.

## 전략 선택 이유

수많은 트레이딩 전략이 있지만, 자동매매에서 중요한 건:
1. **단순함** - 복잡한 전략은 과최적화 위험
2. **검증된 방식** - 모멘텀은 수십 년간 검증됨
3. **명확한 리스크 관리** - 진입보다 청산이 더 중요

그래서 **모멘텀 + 변동성 브레이크아웃** 전략을 선택했다.

## 시스템 구조

```
config.py        # 설정 (심볼, 레버리지, 파라미터)
strategy.py      # 전략 로직 (EMA, RSI, ATR)
risk_manager.py  # 리스크 관리 (포지션 사이징, 일일 손실 제한)
trader.py        # 바이낸스 API 연동
bot.py           # 메인 루프
backtest.py      # 백테스트
```

## 전략 상세

### 진입 조건
- **롱**: EMA(9) > EMA(21), RSI < 70, 볼륨 > 평균의 1.2배
- **숏**: EMA(9) < EMA(21), RSI > 30, 볼륨 > 평균의 1.2배
- MACD 히스토그램으로 신뢰도 보정

### 청산
- **스탑로스**: 2 ATR (변동성 기반 손절)
- **테이크프로핏**: 3 ATR (1:1.5 손익비)
- **트레일링 스탑**: 수익 발생 시 1.5 ATR로 추적

### 리스크 관리
- 거래당 계좌의 **1%** 리스크
- 일일 최대 손실 **3%** 도달 시 거래 중단
- 최대 동시 포지션 **2개**

## 백테스트 결과 (30일)

| 심볼 | 수익률 | 승률 | Profit Factor | 최대 드로다운 |
|------|--------|------|---------------|--------------|
| BTC | +9.58% | 40.4% | 1.14 | 10.82% |
| ETH | +5.23% | 33.5% | 1.08 | 11.46% |

승률이 40% 밖에 안 되지만, 평균 수익($113)이 평균 손실($67)보다 커서 전체적으로 수익이 난다. 이게 **손익비**의 힘이다.

## 배운 점

1. **승률보다 손익비가 중요하다** - 40% 승률도 손익비가 좋으면 수익
2. **ATR 기반 스탑이 효과적** - 고정 퍼센트보다 변동성 적응이 좋음
3. **볼륨 필터가 노이즈 제거** - 거래량 없는 신호는 가짜가 많음
