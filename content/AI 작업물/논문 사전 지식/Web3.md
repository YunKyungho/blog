## 정의
Web3는 [[블록체인]] 기술을 기반으로 한 분산형 인터넷의 차세대 버전으로, 사용자에게 데이터 소유권과 통제권을 부여하는 것을 목표로 한다.

## 웹의 진화
- **Web 1.0**: 읽기 전용 정적 웹 (1990년대)
- **Web 2.0**: 읽기/쓰기 가능한 소셜 웹 (2000년대~현재)
- **Web 3.0**: 읽기/쓰기/소유가 가능한 분산형 웹

## 핵심 기술
- [[블록체인]]: 분산 원장 기술
- [[스마트 컨트랙트]]: 자동 실행 계약
- [[암호화폐]]: 토큰 경제
- 분산 저장소 (IPFS)
- 분산 신원 확인 (DID)

## 특징
- 탈중앙화: 중앙 서버 없이 P2P 네트워크로 운영
- 무신뢰성: 중개자 없이 거래 가능
- 투명성: 모든 거래가 공개적으로 검증 가능
- 사용자 주권: 개인 데이터의 소유권 보장

## 활용 사례
- 분산 금융 (DeFi)
- NFT (대체 불가능 토큰)
- 분산 자율 조직 (DAO)
- [[메타버스]] 플랫폼

## 도전 과제
- 확장성 문제
- 사용자 경험
- 규제 불확실성
- 에너지 소비
