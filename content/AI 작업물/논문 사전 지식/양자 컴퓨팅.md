# 양자 컴퓨팅

**양자 컴퓨팅(Quantum Computing)**은 양자역학의 원리를 이용하여 정보를 처리하는 컴퓨팅 패러다임이다.

## 핵심 개념

### 큐비트(Qubit)
- 고전 컴퓨터의 비트(0 또는 1)와 달리, **중첩(Superposition)** 상태로 0과 1을 동시에 표현
- n개의 큐비트로 2^n개의 상태를 동시에 표현 가능

### 얽힘(Entanglement)
- 두 개 이상의 큐비트가 강하게 상관되어, 하나의 측정이 다른 큐비트에 즉시 영향

### 양자 간섭(Quantum Interference)
- 확률 진폭의 간섭을 통해 원하는 결과의 확률을 증폭

## 주요 알고리즘

- **Shor 알고리즘**: 소인수분해를 다항 시간에 수행 → RSA, ECDSA 위협
- **Grover 알고리즘**: 검색 문제를 √N 시간에 수행 → 대칭키 암호 키 길이 2배 필요

## 암호학적 영향

양자 컴퓨터가 충분히 발전하면 현재의 공개키 암호 체계가 위협받아, [[포스트퀀텀 암호]]로의 전환이 필요하다.

## 양자 블록체인

양자 특성을 활용한 새로운 [[블록체인]] 설계:
- **양자 키 분배(QKD)**: 물리적으로 안전한 키 교환
- **양자 랜덤 넘버**: 진정한 무작위성 제공
