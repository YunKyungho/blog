## 정의

지식 그래프(Knowledge Graph)는 현실 세계의 개체(entity)들과 그들 간의 관계(relationship)를 그래프 구조로 표현한 지식 베이스이다. 노드는 개체를, 엣지는 관계를 나타낸다.

## 상세 설명

### 핵심 구성요소

1. **개체(Entity)**: 사람, 장소, 사물, 개념 등
2. **관계(Relationship)**: 개체 간의 연결 (예: "영향을 미친다", "속한다")
3. **속성(Attribute)**: 개체의 특성 (예: 버전, 날짜)
4. **트리플(Triple)**: (주어, 술어, 목적어) 형태의 기본 단위

### 지식 그래프 표현

```
(CVE-2024-1234) --[영향]-- (Apache Log4j)
(CVE-2024-1234) --[CVSS]-- (9.8)
(CVE-2024-1234) --[유형]-- (원격코드실행)
```

### 사이버보안 응용

- **취약점 분석**: CVE, CWE, CPE 간 관계 표현
- **위협 인텔리전스**: 공격자, 공격기법, 피해자 관계 매핑
- **공격 경로 분석**: 취약점 연쇄를 통한 공격 시나리오 도출
- **보안 지식 통합**: 여러 보안 정보 소스 연결

### NVD 기반 지식 그래프

NVD의 텍스트 설명에서 자동으로 엔티티와 관계를 추출하여 구조화된 지식 그래프 구축 가능:
- 취약점 → 영향받는 제품
- 취약점 → 공격 유형
- 취약점 → 영향 (기밀성/무결성/가용성)

### 구축 기술

- **NLP/NER**: 텍스트에서 엔티티 추출
- **관계 추출**: 엔티티 간 관계 식별
- **온톨로지**: 도메인 지식 체계화
- **임베딩**: 그래프 표현 학습
