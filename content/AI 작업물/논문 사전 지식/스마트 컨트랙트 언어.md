# 스마트 컨트랙트 언어 (Smart Contract Language, SCL)

## 정의
스마트 컨트랙트 언어는 [[블록체인]]에서 실행되는 [[스마트 컨트랙트]]를 작성하기 위해 특별히 설계된 프로그래밍 언어이다.

## 분류
### 범용성에 따라
- **도메인 특화 언어(DSL)**: 스마트 컨트랙트 전용 (Solidity, Vyper)
- **범용 언어(GPL)**: 일반 프로그래밍도 가능 (Go, Rust)

### 추상화 수준에 따라
- **고수준 언어**: 사람이 읽기 쉬움 (Solidity)
- **중간 언어(IR)**: 최적화용 (Yul)
- **저수준 언어**: 기계어에 가까움 (EVM 바이트코드)

## 대표적인 언어들
| 언어 | 플랫폼 | 특징 |
|------|--------|------|
| Solidity | Ethereum | 가장 널리 사용, JavaScript 유사 문법 |
| Vyper | Ethereum | Python 유사, 보안 중심 설계 |
| Michelson | Tezos | 스택 기반, 형식 검증 용이 |
| Plutus | Cardano | Haskell 기반, 함수형 |
| Move | Sui/Aptos | 자원 중심 설계 |

## 설계 목표
1. **사용성**: 개발자가 쉽게 배우고 사용
2. **보안**: 재진입 공격 등 취약점 방지
3. **안전성**: 런타임 오류 최소화
4. **최적화**: 가스 비용 절감
5. **모듈성**: 코드 재사용 용이
