# 합성곱 신경망 (CNN, Convolutional Neural Network)

## 정의

합성곱 신경망은 이미지와 같은 격자 구조 데이터를 처리하기 위해 설계된 [[딥러닝]] 아키텍처이다. **합성곱 연산**을 통해 공간적 계층 구조를 학습한다.

## 핵심 구성 요소

### 1. 합성곱 층 (Convolutional Layer)
- 필터(커널)가 입력을 스캔하며 특징 맵 생성
- 파라미터 공유로 효율적 학습

```
입력 이미지    필터(3x3)     특징 맵
┌───────┐     ┌───┐        ┌─────┐
│ · · · │  *  │ · │   =    │ · · │
│ · · · │     │ · │        │ · · │
│ · · · │     └───┘        └─────┘
└───────┘
```

### 2. 풀링 층 (Pooling Layer)
- 다운샘플링으로 공간 크기 축소
- Max Pooling, Average Pooling

### 3. 완전연결 층 (Fully Connected Layer)
- 최종 분류/회귀를 위한 층
- 모든 뉴런이 연결됨

## 대표 아키텍처

| 모델 | 연도 | 특징 |
|------|------|------|
| **LeNet-5** | 1998 | 초기 CNN, 손글씨 인식 |
| **AlexNet** | 2012 | ImageNet 우승, 딥러닝 부흥 |
| **VGGNet** | 2014 | 3x3 필터, 깊은 네트워크 |
| **ResNet** | 2015 | 잔차 연결, 매우 깊은 학습 가능 |
| **Inception** | 2015 | 다양한 필터 크기 병렬 처리 |

## 응용 분야

- **이미지 분류**: ImageNet, CIFAR
- **객체 탐지**: YOLO, Faster R-CNN
- **세그멘테이션**: U-Net, Mask R-CNN
- **얼굴 인식**: FaceNet
- **[[의료 영상]]**: 진단 보조

## CNN vs 완전연결 네트워크

| 특성 | CNN | 완전연결 |
|------|-----|---------|
| 파라미터 수 | 적음 (공유) | 많음 |
| 공간 정보 | 보존 | 손실 |
| 이미지 처리 | 효율적 | 비효율적 |

## 수용 영역 (Receptive Field)

- 출력 뉴런이 "볼 수 있는" 입력 영역
- 층이 깊어질수록 수용 영역 확대
- 전역적 패턴 인식 가능
