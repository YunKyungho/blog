## 정의
멀티스텝 추론(Multi-Step Reasoning)은 복잡한 문제를 해결하기 위해 여러 단계의 중간 추론 과정을 거치는 인지 능력이다. [[대규모 언어 모델]]에서는 Chain-of-Thought(CoT) 프롬프팅으로 이를 유도한다.

## 상세 설명

### 추론 유형
1. **수학적 추론**: 단계별 계산
2. **논리적 추론**: 전제에서 결론 도출
3. **상식 추론**: 일상 지식 활용
4. **다중 홉 추론**: 여러 정보 조각 연결

### 프롬프팅 기법
- **Zero-shot CoT**: "단계별로 생각해봐"
- **Few-shot CoT**: 예시와 함께 제공
- **Self-Consistency**: 여러 경로 샘플링 후 투표
- **Tree of Thoughts**: 트리 구조 탐색

### LLM 내부 메커니즘
1. **암묵적 추론**: 히든 레이어에서 단계 처리
2. **명시적 추론**: 텍스트로 중간 단계 출력
3. **어텐션 패턴**: 관련 정보 간 연결
4. **레이어별 처리**: 점진적 정보 통합

### Test-Time Compute
- 추론 시간에 더 많은 계산 투입
- o1, DeepSeek-R1의 핵심 아이디어
- 성능 향상 vs 비용 트레이드오프

### 연구 방향
- 추론 경로 해석
- 오류 탐지 및 수정
- 추론 능력 향상 학습
- 효율적 추론 방법
