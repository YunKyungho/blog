# 탈중앙화 금융 (DeFi)

**탈중앙화 금융**(Decentralized Finance, DeFi)은 [[블록체인]] 기술을 활용하여 중앙 기관 없이 금융 서비스를 제공하는 생태계이다.

## 핵심 개념

### 정의
- 전통 금융의 중개자(은행, 증권사 등)를 [[스마트 컨트랙트]]로 대체
- 누구나 접근 가능한 개방형 금융 시스템
- 투명하고 검증 가능한 거래 기록

### 주요 서비스 유형
1. **탈중앙화 거래소(DEX)**: 중앙 거래소 없이 암호화폐 교환 (예: Uniswap, SushiSwap)
2. **대출/차입 프로토콜**: 담보 기반 대출 서비스 (예: Aave, Compound)
3. **스테이블코인**: 가격 안정성을 제공하는 암호화폐
4. **유동성 풀**: 자산을 예치하고 수수료를 얻는 구조
5. **이자 농사(Yield Farming)**: 여러 프로토콜을 활용한 수익 극대화

### 특징
- **무허가성**: 누구나 참여 가능
- **조합성**: 여러 프로토콜을 레고처럼 결합 가능
- **투명성**: 모든 거래와 코드가 공개
- **자기 주권**: 개인이 자산을 직접 관리

## 기술 기반

### 스마트 컨트랙트
- 자동 실행되는 계약 코드
- 주로 이더리움 기반 (Solidity 언어)
- 조건 충족 시 자동으로 거래 실행

### 주요 플랫폼
- **이더리움**: 가장 큰 DeFi 생태계
- **바이낸스 스마트 체인(BSC)**: 낮은 수수료
- **솔라나, 아발란체**: 높은 처리량

## 위험 요소

### 기술적 위험
- 스마트 컨트랙트 취약점
- 오라클 조작
- 플래시 론 공격

### 경제적 위험
- 청산 위험
- 무상 손실(Impermanent Loss)
- 시장 변동성
