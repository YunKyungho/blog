# 딥러닝 (Deep Learning)

## 정의

딥러닝은 **다층 신경망**을 사용하여 데이터에서 계층적 표현(hierarchical representation)을 학습하는 [[머신러닝]]의 하위 분야이다. "딥"은 네트워크의 많은 층(layer)을 의미한다.

## 특징

### 1. 자동 특징 학습
- 전통적 ML: 수동으로 특징 설계 필요
- 딥러닝: 데이터에서 자동으로 특징 학습

### 2. 계층적 표현
- 하위 층: 엣지, 텍스처 등 저수준 특징
- 상위 층: 객체, 개념 등 고수준 특징

### 3. 대규모 데이터 활용
- 데이터가 많을수록 성능 향상
- 소규모 데이터에서는 과적합 위험

## 주요 아키텍처

| 아키텍처 | 용도 | 특징 |
|----------|------|------|
| [[합성곱 신경망]] (CNN) | 이미지 | 공간적 계층 학습 |
| RNN/LSTM | 시퀀스 | 시간적 의존성 |
| Transformer | NLP, 비전 | 어텐션 메커니즘 |
| GAN | 생성 | 적대적 학습 |
| Autoencoder | 압축/생성 | 비지도 학습 |

## 학습 과정

```
입력 → 순전파 → 손실 계산 → 역전파 → 가중치 업데이트 → 반복
```

### 핵심 요소
- **손실 함수**: 예측과 정답의 차이 측정
- **옵티마이저**: SGD, Adam 등
- **역전파**: 그래디언트 계산으로 가중치 업데이트

## 성공 사례

| 분야 | 응용 |
|------|------|
| 컴퓨터 비전 | 이미지 분류, 객체 탐지 |
| 자연어 처리 | 번역, 챗봇, 요약 |
| 음성 | 음성 인식, TTS |
| 게임 | AlphaGo, OpenAI Five |
| [[의료 영상]] | 진단 보조 |

## 한계

1. **대량의 데이터** 필요
2. **계산 비용** 높음 (GPU 필수)
3. **해석 가능성** 부족 (블랙박스)
4. **적대적 공격**에 취약
