## 정의

**추상 구문 트리(Abstract Syntax Tree, AST)**는 소스 코드의 구문 구조를 트리 형태로 표현한 자료구조이다. 코드의 논리적 구조만 남기고 세부 문법 요소(괄호, 세미콜론 등)는 제거한다.

## 핵심 개념

### 구조
```
함수 선언
├── 이름: "add"
├── 매개변수: [a, b]
└── 본문
    └── 반환문
        └── 이항연산 (+)
            ├── 변수 참조: a
            └── 변수 참조: b
```

### 활용 분야
- **컴파일러**: 구문 분석 후 중간 표현으로 사용
- **정적 분석**: 코드 패턴 탐지
- **리팩토링**: 코드 구조 변환
- **AI 기반 코드 분석**: 코드를 구조화된 형태로 모델에 입력

### 장점
- 언어별 문법 차이 추상화
- 코드의 계층적 구조 파악
- 패턴 매칭에 용이
