## 정의

**프로그램 슬라이싱(Program Slicing)**은 프로그램에서 특정 변수나 문장에 영향을 주거나 받는 코드 부분만 추출하는 기법이다.

## 핵심 개념

### 슬라이스 유형
- **후방 슬라이스(Backward Slice)**: 특정 지점에 영향을 주는 모든 문장
- **전방 슬라이스(Forward Slice)**: 특정 지점이 영향을 미치는 모든 문장
- **조건 슬라이스**: 특정 조건 하의 슬라이스

### 활용 분야
- **디버깅**: 버그에 관련된 코드만 분석
- **취약점 분석**: 취약 지점에 영향을 주는 코드 추출
- **코드 이해**: 대규모 코드의 일부분만 파악
- **[[딥러닝]] 입력**: 관련 코드만 모델에 입력

### 예시
```python
x = input()      # 슬라이스 포함
y = 10           # 슬라이스 제외
z = x + 5        # 슬라이스 포함
print(z)         # 슬라이싱 기준점
```

## 취약점 탐지 활용

- 취약한 함수 호출까지의 데이터 흐름 추출
- [[CVE]] 분석 시 관련 코드만 집중 검토
- 모델 입력 크기 감소로 효율성 향상
