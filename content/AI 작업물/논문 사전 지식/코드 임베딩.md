## 정의

**코드 임베딩(Code Embedding)**은 소스 코드를 고정 크기의 수치 벡터로 변환하는 기법이다. 이를 통해 코드를 머신러닝 모델에 입력할 수 있다.

## 핵심 개념

### 표현 방식
- **토큰 기반**: 코드를 토큰 시퀀스로 처리
- **[[추상 구문 트리]] 기반**: 트리 구조 임베딩
- **[[그래프 신경망]] 기반**: CFG/DFG를 그래프로 처리
- **하이브리드**: 여러 표현 결합

### 주요 모델
- **CodeBERT**: 자연어와 코드 동시 학습
- **GraphCodeBERT**: 데이터 흐름 그래프 활용
- **CodeT5**: 코드-코드 변환에 특화
- **StarCoder**: 대규모 코드 생성 모델

### 활용 분야
- 코드 유사도 측정
- 취약점 탐지
- 코드 검색
- 코드 요약/생성

## 취약점 탐지 활용

취약한 코드의 패턴을 임베딩 공간에서 학습:
- 유사한 취약점은 가까운 벡터
- [[CVE]] 패턴 군집화 가능
