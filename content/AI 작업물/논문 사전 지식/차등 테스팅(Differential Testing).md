# 차등 테스팅 (Differential Testing)

## 정의

동일한 기능을 구현한 여러 시스템에 같은 입력을 주고, 출력이 다르면 버그로 판단하는 테스팅 기법.

## 원리

```
입력 X → 시스템 A → 출력 Y1
입력 X → 시스템 B → 출력 Y2
입력 X → 시스템 C → 출력 Y3

Y1 ≠ Y2 ≠ Y3 → 누군가 버그!
```

## 장점

- **오라클 문제 해결**: 정답을 몰라도 "다르다"는 걸로 버그 탐지 가능
- **자동화 용이**: 비교만 하면 됨

## 블록체인에서의 활용

이더리움은 여러 [[블록체인 클라이언트]]가 존재. EthCRAFT는 5개 클라이언트의 [[RPC]] 응답을 비교해서 버그를 찾는다.
