## 정의

뉴로-심볼릭 AI(Neuro-Symbolic AI)는 딥러닝(신경망)의 패턴 인식 능력과 기호적 AI(논리, 규칙)의 추론 및 설명 가능성을 결합한 하이브리드 인공지능 접근법이다.

## 상세 설명

### 두 패러다임의 특징

**신경망 (Neural):**
- 대규모 데이터에서 패턴 학습
- 비정형 데이터 처리에 강점
- 블랙박스적 특성
- 새로운 공격에 적응 가능

**기호적 AI (Symbolic):**
- 명시적 규칙과 논리 기반
- 추론 과정 설명 가능
- 도메인 지식 인코딩
- 적은 데이터로도 작동

### 통합 방식

1. **신경→기호**: 신경망 출력을 기호 시스템에 입력
2. **기호→신경**: 기호 지식으로 신경망 학습 안내
3. **병렬 통합**: 두 시스템이 협력적으로 작동
4. **하이브리드 아키텍처**: 단일 시스템에서 두 요소 결합

### 사이버보안 응용

**장점:**
- **설명 가능한 탐지**: 왜 악성으로 판단했는지 설명
- **제로데이 공격 대응**: 패턴+규칙으로 미지의 위협 탐지
- **도메인 지식 활용**: 보안 전문가 지식 통합
- **강건성 향상**: 적대적 공격에 더 강함

**활용 분야:**
- 악성코드 분류 및 설명
- 침입 탐지 시스템
- 취약점 분석
- 위협 인텔리전스

### 과제

- 신경망과 기호 시스템 간 인터페이스 설계
- 두 표현 간 지식 변환
- 계산 복잡도 관리
