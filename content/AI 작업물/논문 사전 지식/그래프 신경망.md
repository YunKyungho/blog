## 정의

그래프 신경망(Graph Neural Network, GNN)은 그래프 구조의 데이터를 처리하기 위해 설계된 딥러닝 모델이다. 노드(node), 엣지(edge), 그래프 전체의 특징을 학습하여 분류, 예측, 생성 등의 작업을 수행한다.

## 상세 설명

### 기본 원리

GNN은 메시지 전달(message passing) 메커니즘을 통해 작동한다:
1. 각 노드가 이웃 노드로부터 정보를 수집
2. 수집된 정보를 집계(aggregate)하여 자신의 표현을 업데이트
3. 여러 층을 거쳐 그래프 전체의 구조 정보 학습

### 주요 GNN 종류

- **GCN (Graph Convolutional Network)**: 스펙트럴 기반 합성곱 연산
- **GraphSAGE**: 샘플링 기반 이웃 집계
- **GAT (Graph Attention Network)**: 주의(attention) 메커니즘 활용
- **이기종 그래프 신경망 (Heterogeneous GNN)**: 여러 유형의 노드와 엣지 처리

### 사이버보안 응용

- **이상 탐지**: 네트워크 트래픽 패턴에서 비정상 행위 식별
- **내부자 위협 탐지**: 사용자 행동 그래프 분석
- **악성코드 탐지**: 프로그램 호출 그래프 분석
- **접근 권한 분석**: 권한 관계 그래프에서 이상 탐지
