# 스마트 컨트랙트 (Smart Contract)

## 정의

블록체인 위에서 자동으로 실행되는 프로그램. 조건이 충족되면 제3자 없이 계약이 실행된다.

## 작동 원리

```
조건 충족 → 자동 실행 → 결과 블록체인에 기록
```

예: "A가 B에게 1 ETH를 보내면, B는 A에게 NFT를 전송한다"

## 대표 플랫폼

| 플랫폼 | 언어 |
|--------|------|
| Ethereum | Solidity |
| Solana | Rust |
| Cardano | Haskell |

## 보안 취약점

### 재진입 공격 (Reentrancy)
외부 호출 중 상태가 업데이트되기 전에 다시 호출.
- **The DAO 해킹 (2016)**: $60M 손실

### 정수 오버플로우
```solidity
uint8 x = 255;
x = x + 1;  // x = 0 (오버플로우!)
```

### 접근 제어 미흡
권한 체크 없는 함수에 누구나 접근 가능.

## 감사 (Audit)의 중요성

스마트 컨트랙트는 배포 후 수정이 어렵다. 그래서 배포 전 **보안 감사**가 필수.
