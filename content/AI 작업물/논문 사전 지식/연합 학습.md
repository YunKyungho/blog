## 정의
연합 학습(Federated Learning)은 데이터를 중앙 서버로 모으지 않고, 각 분산된 디바이스나 서버에서 로컬 모델을 학습한 후 모델 파라미터만 공유하여 전역 모델을 개선하는 분산 기계학습 패러다임이다.

## 상세 설명

### 동작 원리
1. 중앙 서버가 초기 모델 배포
2. 각 클라이언트가 로컬 데이터로 모델 학습
3. 학습된 모델 파라미터(가중치)만 서버로 전송
4. 서버에서 파라미터 집계(Aggregation)
5. 업데이트된 글로벌 모델 재배포

### 주요 알고리즘
- **FedAvg**: 가장 기본적인 연합 평균 알고리즘
- **FedProx**: 이질성 처리 개선
- **FedOpt**: 서버 측 최적화 적용
- **Scaffold**: 클라이언트 드리프트 보정

### 핵심 도전 과제
1. **데이터 이질성(Non-IID)**: 클라이언트별 데이터 분포 차이
2. **시스템 이질성**: 디바이스 성능 차이
3. **통신 효율성**: 대역폭 제한
4. **보안/프라이버시**: 공격 방어

### 응용 분야
- **모바일 키보드**: 다음 단어 예측
- **의료**: 병원 간 협력 학습
- **금융**: 사기 탐지
- **IoT**: 엣지 디바이스 학습
