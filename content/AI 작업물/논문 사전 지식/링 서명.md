# 링 서명 (Ring Signature)

## 정의
링 서명은 그룹 구성원 중 누군가가 서명했다는 것은 증명하지만, 정확히 누가 서명했는지는 숨기는 디지털 서명 방식이다. 2001년 Rivest, Shamir, Tauman이 제안했다.

## 작동 원리
1. 서명자는 공개키가 알려진 여러 사용자를 선택하여 "링"을 구성
2. 자신의 개인키와 다른 구성원들의 공개키를 사용하여 서명 생성
3. 검증자는 서명이 링 구성원 중 하나에 의해 생성되었음만 확인 가능
4. 실제 서명자의 신원은 숨겨짐

## 핵심 속성
- **익명성**: 링 구성원 중 실제 서명자 식별 불가
- **자발성**: 다른 구성원의 협력이나 동의 불필요
- **연결 불가능성**: 두 서명이 같은 사람에 의한 것인지 판별 불가

## 블록체인에서의 활용
### Monero
- 송금자의 실제 입력을 다른 미끼 입력들과 섞어 익명화
- **RingCT(Ring Confidential Transaction)**: 송금 금액까지 숨김

### 믹싱 서비스
- Möbius 등의 탈중앙화 믹서에서 자금 출처 숨김에 활용

## 장점과 한계
- **장점**: 그룹 구성이 쉽고, 다른 구성원 협력 불필요
- **한계**: 링 크기가 작으면 익명성 감소, 트랜잭션 크기 증가
