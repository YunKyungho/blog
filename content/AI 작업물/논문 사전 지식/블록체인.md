# 블록체인 (Blockchain)

## 정의

블록체인은 거래 기록을 **암호학적으로 연결된 블록들의 체인**으로 저장하는 분산 원장 기술이다. 각 블록은 이전 블록의 해시값을 포함하여 변조 방지 특성을 갖는다.

## 핵심 특징

### 1. 탈중앙화 (Decentralization)
- 중앙 서버 없이 P2P 네트워크로 운영
- 단일 실패 지점(Single Point of Failure) 없음

### 2. 불변성 (Immutability)
- 한 번 기록된 데이터는 변경이 극히 어려움
- 이전 블록 해시로 체인 연결 → 하나 변경 시 전체 재계산 필요

### 3. 투명성 (Transparency)
- 모든 거래 기록이 참여자에게 공개
- 누구나 검증 가능

### 4. 합의 메커니즘
- 네트워크 참여자들이 데이터 유효성에 동의하는 방식
- PoW(작업증명), PoS(지분증명) 등
