# EVM (이더리움 가상 머신)

## 정의
EVM(Ethereum Virtual Machine)은 [[이더리움]] 네트워크에서 [[스마트 컨트랙트]]를 실행하는 분산 컴퓨팅 환경이다. 튜링 완전한 가상 머신으로, 모든 이더리움 노드에서 동일하게 동작한다.

## 핵심 특징
1. **튜링 완전성**: 이론적으로 모든 계산 가능한 문제를 해결할 수 있음
2. **샌드박스 환경**: 스마트 컨트랙트가 네트워크, 파일 시스템, 다른 프로세스에 접근 불가
3. **결정론적 실행**: 동일한 입력에 대해 항상 동일한 출력 보장
4. **가스 시스템**: 무한 루프 방지 및 자원 사용 제한

## 작동 방식
1. 스마트 컨트랙트 코드(Solidity 등)를 EVM 바이트코드로 컴파일
2. 바이트코드를 블록체인에 배포
3. 트랜잭션이 컨트랙트를 호출하면 모든 노드가 동일하게 실행
4. 실행 결과에 대해 합의 도달

## 가스(Gas)
- 각 연산에 고정된 가스 비용이 할당됨
- 복잡한 연산 = 더 많은 가스 소모
- 가스 한도 초과 시 트랜잭션 취소 (Out of Gas)

## EVM 호환 체인
Polygon, BSC, Avalanche 등 많은 블록체인이 EVM과 호환되어 이더리움용 스마트 컨트랙트를 그대로 실행 가능
