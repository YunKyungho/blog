## 한줄 요약
[[딥러닝]]과 AI가 재료 과학 및 공학 분야에서 어떻게 활용되고 있는지, 현재 현황과 핵심 도전과제, 미래 발전 방향을 포괄적으로 분석한 서베이 논문이다.

## 핵심 내용
재료 과학은 현대 기술 발전의 핵심 기반이며, AI의 도입으로 새로운 재료의 발견과 설계가 가속화되고 있다. 이 논문은 AI가 재료 과학에 미치는 영향을 종합적으로 조망한다.

### AI의 역할
1. **재료 발견**: 방대한 화학 공간 탐색
2. **특성 예측**: 실험 없이 재료 특성 추정
3. **합성 최적화**: 제조 공정 개선
4. **구조 분석**: 현미경 이미지 자동 분석

### 핵심 기술
- [[그래프 신경망]]: 분자/결정 구조 표현
- [[트랜스포머]]: 시퀀스 기반 재료 모델링
- 생성 모델: 새로운 재료 구조 생성
- 전이 학습: 제한된 데이터 극복

### 도전 과제
- 데이터 희소성 및 품질 문제
- 해석 가능성 부족
- 실험 검증의 어려움
- 다중 스케일 모델링

## 주요 개념
- [[딥러닝]]: 심층 신경망을 활용한 기계학습
- [[그래프 신경망]]: 그래프 구조 데이터를 처리하는 신경망
- [[트랜스포머]]: 어텐션 메커니즘 기반 신경망 아키텍처

## 입문자를 위한 학습 포인트
1. **재료 인포매틱스**: 데이터 기반 재료 연구의 새로운 패러다임
2. **역설계**: 원하는 특성에서 재료 구조를 찾아가는 접근법
3. **데이터베이스의 중요성**: Materials Project 등 공개 데이터의 역할
4. **시뮬레이션과 ML의 결합**: 전통적 계산과학과 AI의 시너지
