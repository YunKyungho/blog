## 한줄 요약

[[블록체인]]에서 [[암호화폐]] 거래를 추적하는 다양한 기법들을 체계적으로 분류하고, 특히 [[크로스체인]] 환경에서의 추적 과제를 분석한 서베이 논문이다.

## 핵심 내용

### 추적의 필요성
- 범죄 자금(랜섬웨어, 다크웹, 사기) 추적
- 규제 준수(AML, KYC)
- 세금 관련 암호화폐 거래 파악
- 해킹 피해 자금 회수

### 추적 기법 분류

#### 1. 휴리스틱 기반 클러스터링
- **공동 입력 휴리스틱**: 같은 트랜잭션의 입력 주소는 같은 소유자
- **변화 주소 휴리스틱**: 거스름돈 주소 식별
- **시간 분석**: 트랜잭션 타이밍 패턴 분석
- 단점: 휴리스틱 회피 기법 존재

#### 2. 그래프 분석
- 트랜잭션 그래프 구축 (노드=주소, 엣지=송금)
- 중심성 분석, 커뮤니티 탐지
- 자금 흐름 시각화 및 패턴 인식

#### 3. 머신러닝/딥러닝
- 주소 분류 (거래소, 믹서, 도박 등)
- [[그래프 신경망]] 기반 관계 학습
- 이상 탐지 및 패턴 매칭

#### 4. 크로스체인 추적
- 여러 체인에 걸친 자금 흐름 추적
- 브릿지, DEX를 통한 체인 이동 추적
- **핵심 과제**: 범죄자들이 크로스체인으로 세탁 시도

### 추적의 한계

| 장애물 | 설명 |
|--------|------|
| [[믹싱 서비스]] | CoinJoin, Tornado Cash 등 |
| 프라이버시 코인 | Monero, Zcash |
| DEX | 중앙화 KYC 없이 교환 |
| 체인 호핑 | 여러 체인 경유로 추적 어렵게 |

### 연구 방향
- 크로스체인 추적 기술 고도화
- 프라이버시 기술 대응 방법
- 실시간 추적 시스템
- 법적 증거 능력 확보

## 주요 개념

- [[트랜잭션 추적]]: 블록체인에서 자금 흐름을 분석하는 기술
- [[믹싱 서비스]]: 암호화폐 익명성 강화 서비스
- [[크로스체인]]: 서로 다른 블록체인 간 연결 기술
- [[블록체인]]: 분산 원장 기술

## 입문자를 위한 학습 포인트

1. **비트코인 추적 가능한가?**: 
   - 모든 거래가 공개되어 있어 기본적으로 추적 가능
   - 하지만 주소와 실제 사람 연결이 어려움
   - 거래소 KYC 기록으로 신원 확인

2. **경찰은 어떻게 추적하나?**:
   - 블록체인 분석 회사(Chainalysis, Elliptic) 도구 사용
   - 알려진 거래소/서비스 주소와 연결
   - 현금화 시점에서 신원 확보

3. **범죄자들의 회피 기법**:
   - 믹서로 자금 섞기
   - 여러 체인 경유 (Bitcoin → Monero → Ethereum)
   - 탈중앙 거래소(DEX)에서 KYC 없이 교환

4. **추적 기술의 발전**:
   - AI/ML로 패턴 인식 강화
   - 크로스체인 분석 도구 개발
   - 실시간 모니터링 시스템

5. **프라이버시 vs 추적**: 
   - 합법적 프라이버시 보호 필요
   - 동시에 범죄 자금 추적도 필요
   - 기술과 규제의 균형 찾기
