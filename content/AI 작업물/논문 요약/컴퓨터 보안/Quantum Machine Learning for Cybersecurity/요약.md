## 한줄 요약

양자 머신러닝(QML) 기술을 사이버보안에 적용하는 방법을 체계적으로 분류하고, 양자 신경망, 양자 SVM 등 주요 QML 알고리즘의 보안 응용을 종합적으로 분석한 서베이 논문.

## 핵심 내용

### 연구 배경
- 사이버 위협의 복잡성 증가로 기존 머신러닝의 한계 대두
- 양자 컴퓨팅이 고차원 데이터 처리에서 잠재적 우위 제공
- 양자역학 기반 계산으로 특정 문제에서 지수적 속도 향상 가능

### 주요 QML 기법
1. **양자 신경망 (QNN)**: 양자 회로 기반 신경망으로 복잡한 패턴 학습
2. **양자 서포트 벡터 머신 (QSVM)**: 고차원 특징 공간에서 효율적 분류
3. **변분 양자 고유값 솔버 (VQE)**: 최적화 문제 해결
4. **양자 근사 최적화 알고리즘 (QAOA)**: 조합 최적화에 활용

### 사이버보안 응용 분야
- 침입 탐지: 네트워크 트래픽 패턴 분석
- 악성코드 분류: 행동 패턴 기반 탐지
- 암호 분석: 취약점 발견
- 이상 탐지: 정상 패턴에서 벗어난 활동 식별

### 분류 체계(Taxonomy)
- 지도학습 기반 QML 보안 응용
- 비지도학습 기반 QML 보안 응용
- 강화학습 기반 QML 보안 응용
- 하이브리드 양자-고전 접근법

## 주요 개념

- [[양자 머신러닝]]: 양자역학 원리를 활용한 머신러닝 기법
- [[딥러닝]]: 심층 신경망 기반 학습
- [[이상 탐지]]: 정상 패턴에서 벗어난 데이터 식별
- [[침입 탐지 시스템]]: 네트워크 보안 모니터링 시스템

## 입문자를 위한 학습 포인트

1. **양자 컴퓨팅 기초 이해**: 큐비트, 양자 중첩, 양자 얽힘의 개념 파악
2. **ML 배경 필요**: 신경망, SVM 등 기본 ML 알고리즘 이해가 선행되어야 함
3. **하이브리드 접근**: 현 시점에서는 양자-고전 하이브리드 방식이 실용적
4. **향후 발전 가능성**: 양자 컴퓨터 발전에 따라 보안 분야 혁신 가능
