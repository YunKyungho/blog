## 한줄 요약

대규모 언어 모델(LLM)이 사이버보안에 미치는 양면적 영향을 분석하여, 방어적 활용(위협 탐지, 취약점 평가)과 LLM 자체의 보안 취약점 및 방어 기법을 종합적으로 다룬 서베이.

## 핵심 내용

### LLM의 사이버보안 활용 (방어적)
1. **위협 탐지**
   - 악성코드 분석 및 분류
   - 피싱 이메일 탐지
   - 네트워크 이상 탐지

2. **취약점 평가**
   - 코드 보안 분석
   - CVE 설명 분석 및 요약
   - 취약점 우선순위화

3. **인시던트 대응**
   - 보안 로그 분석
   - 보고서 자동 생성
   - 대응 조치 권장

4. **보안 교육**
   - 대화형 보안 교육
   - 시뮬레이션 기반 훈련

### LLM의 취약점 (공격 표면)
- **프롬프트 인젝션**: 악의적 입력으로 모델 조작
- **탈옥(Jailbreak)**: 안전 가드레일 우회
- **데이터 유출**: 학습 데이터 추출 공격
- **환각(Hallucination)**: 잘못된 보안 정보 생성
- **모델 역추론**: 모델에서 민감 정보 추출

### LLM 방어 기법
- **입력 필터링**: 악의적 프롬프트 차단
- **출력 검증**: 생성된 내용의 안전성 확인
- **파인튜닝**: 보안 특화 모델 학습
- **레드팀**: 적대적 테스트로 취약점 발견
- **가드레일**: 안전 정책 적용

### 도메인별 활용
- **IoT 보안**: 제한된 환경에서 경량화된 적용
- **블록체인**: 스마트 컨트랙트 분석
- **의료 보안**: HIPAA 준수 분석 지원
