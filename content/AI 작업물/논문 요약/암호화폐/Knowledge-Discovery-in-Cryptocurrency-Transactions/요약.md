## 한줄 요약

[[암호화폐]] 트랜잭션 데이터에서 데이터 마이닝 기법을 활용한 지식 발견 연구를 트랜잭션 추적, 집단 행동 분석, 개인 행동 분석의 세 측면으로 체계적으로 분류한 포괄적 서베이.

## 핵심 내용

### 연구 배경
- [[블록체인]]의 공개적 특성: 모든 트랜잭션 이력 공개
- 풍부한 데이터 = 연구 기회
- 사용자 행동의 전체 스펙트럼 기록
- 217개 참고문헌 분석

### 지식 발견 3대 영역

#### 1. 트랜잭션 추적 및 주소 연결 (Transaction Tracing)

**주소 클러스터링**
- **공동 입력 휴리스틱**: 같은 트랜잭션의 입력 주소는 동일 소유자
- **변경 주소 휴리스틱**: 새로 생성된 출력 주소는 발신자의 거스름돈
- **시간 분석**: 활동 패턴으로 주소 그룹화

**익명화 해제 기법**
- IP 주소 연결
- 거래소 입출금 분석
- 오프체인 데이터 결합

**대응 기술**
- [[트랜잭션 믹싱]]: CoinJoin, 믹서 서비스
- 프라이버시 코인: Monero, Zcash

#### 2. 집단 사용자 행동 분석 (Collective Behaviors)

**시장 분석**
- 가격 변동 패턴
- 거래량과 가격 상관관계
- 버블 및 붕괴 예측

**네트워크 분석**
- 트랜잭션 그래프 구조
- 부의 분포: 고래(대량 보유자) 식별
- 자금 흐름 패턴

**이상 탐지**
- 폰지 스킴 식별
- 자금세탁 탐지
- 사기 주소 분류

#### 3. 개인 사용자 행동 분석 (Individual Behaviors)

**사용자 프로파일링**
- 거래 빈도와 금액 패턴
- 시간대별 활동
- 선호하는 서비스 유형

**엔티티 식별**
- 거래소 주소 태깅
- 마이닝 풀 식별
- 다크넷 마켓 연결

**행동 예측**
- 향후 거래 패턴
- 보유 기간 예측
- 사용자 이탈 분석

### 연구 방법론

**데이터 수집**
- 풀노드 운영: 직접 블록체인 데이터 파싱
- API 활용: Blockchain.info, Etherscan
- 상업 서비스: Chainalysis, Elliptic

**분석 기법**
| 기법 | 적용 영역 |
|------|----------|
| 그래프 분석 | 주소 클러스터링, 네트워크 구조 |
| 머신러닝 | 사기 탐지, 분류 |
| 시계열 분석 | 가격 예측, 행동 패턴 |
| NLP | 소셜 미디어 감성 분석 |

### 시각화 도구
- **GraphSense**: 주소 클러스터 시각화
- **BlockSci**: 대용량 블록체인 분석
- **Chainalysis Reactor**: 상업용 조사 도구

### 미래 연구 방향
- [[탈중앙화 금융]](DeFi) 트랜잭션 분석
- 디지털 법정화폐(CBDC) 데이터
- 크로스체인 분석
- 프라이버시 보존 분석 기법

## 주요 개념
- [[암호화폐]]: 암호학 기반 디지털 화폐
- [[블록체인]]: 분산 원장 기술
- [[트랜잭션 믹싱]]: 프라이버시 보호를 위한 자금 섞기

## 입문자를 위한 학습 포인트

1. **"익명" ≠ "비공개"**: 
   - 블록체인은 가명(pseudonymous)이지 익명이 아님
   - 모든 거래 기록이 영구 공개

2. **데이터 과학 관점**: 
   - 암호화폐 = 풍부한 금융 데이터셋
   - 전통 금융에서 불가능한 분석 가능

3. **실습 추천**: 
   - Etherscan으로 트랜잭션 탐색
   - 간단한 주소 클러스터링 구현

4. **윤리적 고려**: 
   - 프라이버시 vs 투명성 트레이드오프
   - 분석 기술의 양면성 (범죄 추적 vs 감시)

## 논문의 가치

- 도구와 데이터 소스 **실용적 정보** 제공
- DeFi, CBDC 등 **미래 방향** 제시
