## 한줄 요약

[[블록체인]] 암호화폐의 익명화 도구인 [[믹싱 서비스]]의 기술과 구현을 중앙화/탈중앙화, 프라이버시 코인, 크로스체인 등으로 분류하고 19개 주요 서비스를 분석한 포괄적 서베이.

## 핵심 내용

### 믹싱의 필요성
- **블록체인의 역설**: 탈중앙화지만 완전 투명한 공개 원장
- **익명성 착각**: 주소 기반 가명성(pseudonymity)이지 완전한 익명성 아님
- **테인트 분석(Taint Analysis)**: 불법 코인 추적 기술로 일반 사용자 프라이버시도 위협
- **믹싱 목적**: 송금자와 수신자 사이의 연결 끊기

### 믹싱 서비스 분류

#### 1. 중앙화 믹서
| 서비스 | 메커니즘 | 특징 |
|--------|----------|------|
| Helix | 다중 홉 스왑, 깨끗한 코인 | 2017년 압수됨 |
| Bitcoin Fog | 조인 트랜잭션, 무작위 지연 | 2021년 압수됨 |
| Obscuro | [[TEE]] 활용 셔플링 | 하드웨어 의존 |

**문제점**: 신뢰 필요, 법적 압수 취약, 로그 유출 위험

#### 2. 탈중앙화 믹서

##### CoinJoin 계열
- **CoinJoin**: 여러 사용자 트랜잭션을 하나로 결합
- **CoinShuffle**: 셔플링으로 순서 익명화
- **CoinParty**: 임계 암호로 내결함성 향상

##### 이더리움 기반
| 서비스 | 기술 | 특징 |
|--------|------|------|
| Möbius | [[링 서명]] | 그룹 익명성 |
| AMR | zk-SNARK | 보상 인센티브 |
| Tornado Cash | [[영지식 증명]] | 가장 인기, 2022년 제재 |
| MixEth | zk-SNARK | 가스 최적화 |

##### 취약점
- **DoS 공격**: 참여자가 서명 거부 시 전체 중단
- **Sybil 공격**: 공격자가 익명성 집합 다수 장악

#### 3. 프라이버시 코인
| 코인 | 기술 | 프라이버시 수준 |
|------|------|----------------|
| Monero | [[링 서명]], 스텔스 주소, RingCT | 기본 적용 |
| Zcash | zk-SNARK, 차폐 풀 | 선택적 |
| Oasis/Secret | [[TEE]], 기밀 스마트 컨트랙트 | 계약 수준 |
| Integritee/Phala | [[TEE]], 오프체인 실행 | Polkadot 생태계 |

#### 4. 중앙화 거래소 (CEX)
- Binance, Coinbase 등이 간접적 믹싱 효과 제공
- 크로스체인 거래로 추적 복잡화
- 그러나 KYC로 익명성 제한

### 믹싱 기법 (프리미티브)

| 기법 | 설명 | 탐지 회피 |
|------|------|----------|
| 스와핑 | 여러 주소 간 이동 | 중간 |
| 셔플링 | 순서 무작위화 | 높음 |
| 집계 | 다수 입력 하나로 합침 | 중간 |
| 필링 체인 | 점진적 분배 | 낮음 (패턴 노출) |
| 무작위 수수료 | 탐지 회피용 | 높음 |
| 무작위 지연 | 시간 패턴 숨김 | 높음 |
| 새 주소 생성 | 주소 재사용 방지 | 필수 |
| 오프체인 | 블록체인 외부 처리 | 높음 |
| 영지식 증명 | 소유권만 증명 | 매우 높음 |

### 보안 분석
- **핵심 취약점**: 익명성 집합 크기가 작으면 추적 가능
- **시간 분석**: 짧은 입출금 간격이 가장 큰 탈익명화 요인
- **권장사항**: 오래 보관할수록 안전

## 주요 개념
- [[믹싱 서비스]]: 트랜잭션 경로를 숨기는 익명화 도구
- [[영지식 증명]]: 정보 노출 없이 사실 증명
- [[링 서명]]: 그룹 중 누군가의 서명임만 증명
- [[TEE]]: 하드웨어 기반 보안 실행 환경

## 입문자를 위한 학습 포인트

1. **익명성 ≠ 가명성**: 블록체인 주소는 익명이 아닌 가명
2. **익명성 집합 이해**: 같이 섞이는 사용자 수가 프라이버시 결정
3. **시간이 중요**: 입금 후 바로 출금하면 추적 쉬움
4. **Tornado Cash 사례 학습**: 기술적으로 안전하나 법적 리스크 존재
5. **프라이버시 코인 비교**: Monero(기본 프라이버시) vs Zcash(선택적)
