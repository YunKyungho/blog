Access Network는 크게 두 가지로 구분 된다.

1. **홈 접속 네트워크(Home Access Network)**
2. **엔터프라이즈 접속 네트워크(Enterprise Access Network)**

여기서 **홈 접속 네트워크**는 **Residential Access Network** 라고도 불린다.

**엔터프라이즈 접속 네트워크**는 자체적으로 구축과 운영이 가능한 것을 말한다.

Access Network에서 실제로 통신링크를 구성하는 물리적인 매개체(Physical Media)를 짚고 넘어가자.

**물리 매체(Physical Media)**

**네트워크 엣지(Network Edge)**

- 인터넷에서 호스트 또는 종단 장치가 연결되는 접속 네트워크
- 사실 상 네트워크 엣지 = 접속 네트워크 라고 볼 수 있다.

**접속 네트워크 구분**

- Home Access Network : DSL, Cable, FTTH, Ethernet/WiFi
- Enterprise Access Network : Ethernet/WiFi
- Mobile Access Network : 3G, 4G/LTE, 5G

접속 환경에 따라서 Home Access Network는 많은 종류가 있다.

**Home Access Network**

- DSL : (Digital subscriber Line) 전화선을 사용하는 네트워크
- Cable : CA TV에서 사용하는 케이블 모뎀을 사용하는 네트워크
- FTTH : (Fiber To The Home) 광 케이블을 집 까지 연결 해서 사용하는 네트워크
- Ethernet : 유선랜을 사용하는 네트워크
- WiFi : 무선랜을 사용하는 네트워크

**Enterprise Access Network**

- 회사 등 대부분의 경우 랜을 이용하여 Host 장치들을 연결해서 네트워크를 구축한다.

**Mobile Access Network**

- Mobile Access Network 같은 경우 기술 적인 범위가 아주 넓어 컴퓨터 네트워크의 범위에서 벗어난다. 그렇기에 주로 이를 제외한 위 두가지를 다룬다.

**FTTH(Fiber To The Home) 네트워크 구성**

**DSL**의 전화선과 **Cable** 네트워크의 케이블은 **구리선**이지만 **FTTH**는 **광케이블**을 사용한 Access Network이다. **광케이블**이 구리선에 비해 **대역폭이 높기** 때문에 **고속 데이터 통신이 가능**하다. 
따라서 **구축 비용은 많이** 들지만 위 두가지 네트워크 보다 **고속의 인터넷 사용**이 가능하다.

![Untitled](images/Network/Untitled%208.png)

위 사진 처럼 하나의 구역 까지 광케이블을 가져온 뒤 그 구역의 가정들에게 광케이블을 분기 하여 공유하는 형태이다.

이는 광케이블 하나의 전송속도를 여러 가정에서 공유한다는 의미이며 이를 구축하는 방식에도 여러 종류가 있다.

**TDM-PON**

- 광 분기기에 의해 분기된 광 케이블에 연결된 모든 가입자가 시간 분할 다중 접근(TDMA-Time Division Multiple Access)으로 광 케이블의 대역폭을 공유
- 10Gbps 전후의 공유 전송 속도 제공

말 그대로 어느 시간대는 어떤 모뎀을 사용하고 또 다음 시간대는 다음 모뎀이 사용하는 것이다.

(처음에 몇 분 구간으로 나눈다 생각하여 이게 뭔 소린가 싶었는데 아마 몇 초 단위로 나누는게 아닐까 싶다.)

**WDM-PON**

- 파장 분할 다중 접근(Wavelength Division Multiple Access) 기술을 적용하여 각 가입자별로 서로 다른 파장(wavelength)의 광 신호 할당
- 수Gbps 전후의 전용 전송 속도 제공

파장은 주파수가 달라지면 주기가 다르고 주기가 달라지면 파장이 달라지는 특성이 있다.
그렇기에 주파수가 다른 신호를 보냈을 때 수신하는 쪽에서 구분하여 수신이 가능하다.
파장 분할 다중 접근이 이러한 특성을 이용한 방식이다.

각 가입자 별로 서로 다른 파장을 할당 받기에 빠른 전용 전송 속도를 제공 받을 수 있지만 구축 방식이 복잡하기에 구축 비용이 높다.

**Ethernet/WiFi Access Network**

특정 지역에 구성되는 네트워크를 LAN(Local Area Network)라고 하며 아파트의 단지, 동 마다 별도의 **LAN**을 주로 구축하고 이 LAN 가정의 홈 네트워크에 직접 연결하는 구축 방식이다.

이때 주로 사용되는 유선 네트워크가 Ethernet고 무선 네트워크가 WiFi다.

컴퓨터 네트워크 강의에선 자세한 설명은 생략하고 이에 대해 자세한 강의를 듣고 싶다면

**물리 매체(Physical Media)**

**정의**

- 정보를 표현하는 신호를 전달하는 매개체
- 통신 장치의 물리적 연결 매개체

**유형**

- 유도 매체(Guided media) : TP(Twisted Pair), 동축 케이블. 광케이블
- 비유도 매체(Unguided media) : WiFi, 위성, 이동전화 무선 채널

간단히 설명하고 넘어가셨는데 이 역시 자세히 알고 싶다면 네트워크 설계 1강, 2강 / 데이터통신 13강을 들어보라고 하셨다.
