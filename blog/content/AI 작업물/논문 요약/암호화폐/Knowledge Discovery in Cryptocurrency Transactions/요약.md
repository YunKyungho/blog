## 한줄 요약

암호화폐 트랜잭션 데이터에서 데이터 마이닝 기법을 활용하여 지식을 추출하는 연구들을 종합 분석한 서베이이다.

## 핵심 내용

[[암호화폐]]는 전체 생성 및 거래 이력을 공개함으로써 사용자 신뢰를 얻는다. 이 거래 이력은 암호화폐 사용자 행동의 모든 스펙트럼을 충실히 기록한다.

### 연구 분류

1. **트랜잭션 추적 및 주소 연결**
   - 같은 사용자의 여러 주소 클러스터링
   - 휴리스틱 기반 연결
   - 거래소 입출금 추적

2. **집단 사용자 행동 분석**
   - 네트워크 토폴로지 분석
   - 시장 조작 탐지
   - 자금 흐름 패턴

3. **개별 사용자 행동 연구**
   - 사용자 프로파일링
   - 역익명화 시도
   - 행동 예측

### 분석 기법

- 그래프 분석
- 머신러닝/[[딥러닝]]
- 통계적 분석
- 시계열 분석

### 도구 및 서비스

- 블록 파싱 도구
- 시각화 서비스
- 분석 플랫폼

### 미래 방향

- [[탈중앙화 금융]] 분석
- 중앙은행 디지털 화폐(CBDC)
- 크로스체인 분석

## 주요 개념

- [[암호화폐]]: 분석 대상
- [[블록체인]]: 데이터 소스
- [[딥러닝]]: 분석 기법
- [[탈중앙화 금융]]: 새로운 분석 영역

## 입문자를 위한 학습 포인트

1. 블록체인 데이터의 공개성과 분석 가능성 이해
2. 왜 완전한 익명성이 어려운지 파악
3. 데이터 분석이 규제와 보안에 어떻게 활용되는지
