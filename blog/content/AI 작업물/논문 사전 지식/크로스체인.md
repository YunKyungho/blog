## 정의

크로스체인(Cross-chain)은 서로 다른 [[블록체인]] 네트워크 간에 자산, 데이터, 또는 정보를 교환할 수 있게 하는 기술이다. 블록체인 생태계의 파편화 문제를 해결하고 상호운용성을 높이기 위해 개발되었다.

## 상세 설명

### 주요 접근 방식

1. **사이드체인(Sidechains)**: 메인 체인과 연결된 별도의 블록체인
2. **아토믹 스왑(Atomic Swap)**: 중개자 없이 서로 다른 [[암호화폐]] 간 직접 교환
3. **브릿지(Bridge)**: 두 체인을 연결하는 프로토콜
4. **공증인 기반(Notary-based)**: 신뢰할 수 있는 제3자가 거래를 검증
5. **해시 타임락 계약(HTLC)**: 시간 제한과 해시를 이용한 조건부 거래

### 보안 고려사항

크로스체인 브릿지는 해킹의 주요 타겟이 되어왔다. 신뢰 가정, 검증 메커니즘, 자금 보관 방식이 보안에 중요한 영향을 미친다.
